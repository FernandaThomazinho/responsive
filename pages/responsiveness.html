Projetando para a melhor experiência para todos os usuários

Páginas web podem ser visualizadas utilizando diferentes dispositivos: desktops, tablets e smartphones. Mesmo as telas entre os desktops e notebooks podem ter diferentes resoluções e a quantidade e proporção dos pixels que uma tela é capaz de apresentar influem diretamente na aparência de um conteúdo editado por folhas de estilo. Sua página web deve ter uma boa aparência e se simples de ser usada, independentemente do dispositivo que a acessa.

Apresentação do Conteúdo

Páginas web não devem deixar de apresentar informações para caber em dispositivos pequenos, mas ao invés disso, devem adaptar seu conteúdo para caber em qualquer dispositivo. É chamado design web responsive quando se utiliza CSS e HTML para redimensionar, esconder, encolher, ou mover o conteúdo de uma página para fazer com que seja bem apresentado em qualquer tela.

User Experience

Muitos são os itens necessários para se oferecer uma boa experiencia para o usuário de um sistema. Tempo de resposta, facilidade de encontrar o conteúdo buscado, boa navegação e facilidade de aprendizado, são alguns deles. Estar preparado para apresentar seu conteúdo de forma adaptativa ao dispositivo que o acessa é um requisito de sistemas web abertos para o público ou que possuam alguma caracteristica de uso que envolva o uso de dispositivos móveis. A responsividade é a caracteristica dos sistemas que atendem a este requisito.

Sistemas Multiplataforma

São os sistemas que possuem a caracteristica de se manterem os mesmos independentemente da forma de acesso ou da versão que é utilizada. Um exemplo bem simples de sistema multiplataforma é o Facebook. Acessando em qualquer sistema operacional de celular, seja pelo navegador, sendo ele de um mobile, notebook, tablet ou desktop, ou pelo prórpio aplicativo, como usuários, não sentimos a diferença entre os sistemas e conseguimos aprender onde encontrar cada item que possa ser procurado sem muito esforço.
