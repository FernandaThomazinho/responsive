Projetando para a melhor experiência para todos os usuários

Páginas web podem ser visualizadas utilizando diferentes dispositivos: desktops, tablets e smartphones. Mesmo as telas entre os desktops e notebooks podem ter diferentes resoluções e a quantidade e proporção dos pixels que uma tela é capaz de apresentar influem diretamente na aparência de um conteúdo editado por folhas de estilo. Sua página web deve ter uma boa aparência e se simples de ser usada, independentemente do dispositivo que a acessa. Páginas web não devem deixar de apresentar informações para caber em dispositivos pequenos, mas ao invés disso, devem adaptar seu conteúdo para caber em qualquer dispositivo.

É chamado design web responsive quando se utiliza CSS e HTML para redimensionar, esconder, encolher, ou mover o conteúdo de uma página para fazer com que seja bem apresentado em qualquer tela.
